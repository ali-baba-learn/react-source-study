
## 项目技术栈

react@16.5.2 + webpack@4.35.3



## 项目运行


```
git clone --depth 1 https://github.com/ali-baba-learn/react-source-study.git  

cd react-source-study 

npm install || yarn  install

npm run dev


```
构建命令
```

npm run build (打包，部署)

```
